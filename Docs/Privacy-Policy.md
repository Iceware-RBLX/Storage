# Privacy Policy

last updated: april 9 2026

we only collect the minimum data needed to run the script and to protect licenses, nothing extra

## what data is collected

when the script runs, some basic information is logged through our licensing/authentification system **(luarmor)**

this may include

* license key
* hwid (hardware id)
* executor used
* basic execution info (like when the script runs or how many times)
* discord username (linked to buyer keys)

this data is used to make sure everything is working properly, prevent abuse, and help fix problems if something goes wrong

## third party services

we use third party services to function properly,

these include

* luarmor — used for licensing
* work.ink — used for monetization

these services handle data differently, we do not have control over they manage their data

## how data is used

collected data is only used for:

* authentification
* preventing abuse or key sharing
* support and troubleshooting

**we do not sell data to anyone**
