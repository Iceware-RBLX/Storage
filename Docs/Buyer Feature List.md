# **Iceware Premium — Feature List**

## **Script**

- **Permanent Key Access -** Grants you a permanent key, allowing access without needing to complete the key system every 24H or 12H

## **Midnight Chasers**

* **[FE] Fuck up every bike -**
  Causes all players riding bikes to lose stability, making turning difficult and forcing bikes to fall over when idling

## **Driving Empire**

* **Instant Arrest -**
  Instantly arrests criminals, this can generate over 40M+ per hour
